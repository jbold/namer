# Strategic Naming Framework

## Strategic Discovery

Before generating names, work through these five questions with the user. These define the space the name must occupy.

```
       VISION
       (top)
      /      \
  MESSAGE    ADVANTAGE
  (left)     (right)
      \      /
      GAPS
     (bottom)
       +
     DOMAIN
```

1. **Vision** — What is this thing? What does it do, who's it for, and what does success look like?
2. **Advantage** — What makes it different from the competition? What's the moat?
3. **Gaps** — What's missing right now? Early stage, no users, credibility, resources, etc.
4. **Message** — What should the name *feel* like? Serious/scientific? Playful/accessible? Futuristic? Something else?
5. **Domain** — What industry or category is this? Examples: "coffee shop", "npm package", "AI tool", "law firm", "band name", "SaaS platform"

The domain answer is critical for availability checking. A name might be clear for a coffee shop but taken for a SaaS tool. The agent appends the domain to web searches: `"[name] [domain]"`.

Make questions concrete and conversational — adapt to the product context. Avoid abstract strategy language.

## Key Principles

1. **Volume first**: 1000-1500 candidates before evaluation. ~250 worth polishing from ~3000 ideas.
2. **Separate creation from judgment**: Distinct phases — generate (no-filter), speculative discussion ("what could this do for us?"), then rigorous pruning.
3. **Three-team method**: One team knows the real brief. One thinks they're naming a competitor. One works on an unrelated category. Cross-pollination produces unexpected winners.
4. **Great names make you uncomfortable**: If it feels safe, it's probably wrong.
5. **Polarizing = strong signal**: Polarization tells you there's energy. Bland names have no energy.
6. **Competitor test**: Don't ask "what do you think?" — ask "our competitor just launched with this name, what's your reaction?"
7. **Compounds are multipliers**: Two words combined create 1+1=3 mental associations.
8. **Domains don't matter anymore**: AI changed search. Get the right name, solve domain later.

## Sound Symbolism

Research shows certain phonemes carry consistent psychological associations:

| Letter | Effect | Why it works |
|--------|--------|-------------|
| V | Most alive/vibrant | Open, voiced, energetic |
| B | Most reliable | Solid, grounded, stable |
| Z | Draws attention | Uncommon, buzzing, electric |
| X | Signals innovation | Rare, edgy, technical |

## Process

1. **Create** — Wild generation, multiple metaphorical domains, no evaluation
2. **Invent** — Shape and discuss candidates speculatively
3. **Implement** — Legal clearance, linguistic filters, strategy criteria, final selection
